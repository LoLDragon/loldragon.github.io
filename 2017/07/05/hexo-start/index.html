搭建一些注意事项：

1.创建github仓库时，记住仓库的名字命名为：xxx.github.io，这样你生成的github的博客主页网址就可以是：https://lxxx.github.io/

2.创建标签页和分类页时，记住添加一下type:xxx,比如type:’tags’ ;默认这里生成的index.md是么没有的；如果不设置标签页中不会出现对应的标签值；

3.在安装algolia搜索功能时，在采用默认安装hexo-algolia插件时生成的版本为1.1.0时，会出现在hexo algolia上传数据，更新index失败的情况；错误提示为：“Please provide an API key”，这时候改为安装回“npm install hexo-algolia@0.2.0 –save” 0.2.0版本时可以正常更新；
